# AnimeList

## Setup Instructions

1. Clone the repository
2. Install dependencies: `go mod tidy`
3. Start the server: `go run main.go`

## API Endpoint

- **POST /auth/sign-in**
- **POST /auth/sign-up**
- **GET /anime/:id**
- **PUT /anime/:id**
- **DELETE /anime/:id**
- **GET /anime**
- **POST /anime**
